# Telegram Payment Bot

A Telegram bot for processing payments using Razorpay.

## Features
- Create payment orders
- Generate QR codes
- Check payment status
- View payment history

## Deploy on Render
1. Fork this repository
2. Connect to Render
3. Add environment variables
4. Deploy!

## Commands
- `/pay` - Make a payment
- `/history` - View payment history
- `/help` - Show help
